وظایف طراح تجربه کاربری UX چیست؟
انجام تحقیقات کاربر شامل مصاحبه و نظرسنجی با آن‌ها ساخت پرسونای کاربر بر اساس تحقیقات ساخت مسیر کاربر در پیدا کردن آسان راه حل ارائه شده توسط کسب و کار و استفاده از آن ساخت نمونه و پروتوتایپ محصول بر اساس مسیر آزمایش کردن نمونه‌ها توسط کاربران واقعی و بهینه‌سازی آن ارتباط مناسب با دیگر بخش‌های محصول مانند برنامه‌نویس‌ها، طراح محصول و UI
وظایف طراح رابط کاربری UI چیست؟
گرفتن اطلاعات مورد نیاز و نمونه‌های ساخته شده در بخش UX ساخت نمونه‌های اولیه گرافیکی از محصول دیجیتال طراحی المان‌های تصویری شامل رنگ‌ها، دکمه‌ها و منوها طراحی تعاملی مانند انیمیشن‌ها و حرکت‌ها طراحی ریسپانسیو و واکنش گرا مناسب همه دستگاه‌های مورد استفاده کاربر
